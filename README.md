# ScpoWR

幻想私社开发！——个比较小但是用起来还挺方便的小网络请求函数库

内含
- 表单元素、请求字符串和请求键值对相互转换
- 一个函数即可发送网络请求
- 类似ES6`Promise`的炫酷函数式编程异步请求方法

### 使用方法：

1. `<script src="//js.seventop.top/scpo-webreq.js"></script>`
   引入ScpoWR。
2. 用。
